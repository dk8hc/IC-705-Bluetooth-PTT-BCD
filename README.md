# IC-705-BlueTooth-PTT Switch and BCD Output

PTT via Pin 26 (HF VHF UHF)

BCD A	  Pin 33
BCD B	  Pin 32
BCD C 	Pin 22
BCD D	  Pin 17

Short Vid: https://www.youtube.com/shorts/tgXO_a_IPUA

original based on https://github.com/PE1OFO/IC-705-BlueTooth-Controller, used this fork: https://github.com/dk8hc/IC-705-BlueTooth-Controller for binary-coded decimal extension.

You have to add following libraries to your Arduino Soft:

https://github.com/WillyIoBrok/CIVmasterLib

https://github.com/Bodmer/TFT_eSPI

If you use this also for control your Xiegu XPA-125B check bandswitching voltages before using!
